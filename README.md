## 🚀 Deploy to Vercel (Free - 5 Minutes)

### Step 1: Create a GitHub account
Go to https://github.com and sign up (free)

### Step 2: Upload this project
1. Click "New repository" on GitHub
2. Name it: `mindset-assessment`
3. Upload all files from this folder

### Step 3: Deploy on Vercel
1. Go to https://vercel.com
2. Click "Sign up" → choose "Continue with GitHub"
3. Click "New Project"
4. Select your `mindset-assessment` repository
5. Click "Deploy" (no changes needed!)
6. In 60 seconds you get a FREE link like:
   👉 `https://mindset-assessment.vercel.app`

### Step 4: Share with clients!
Send the link via WhatsApp or Instagram DM
